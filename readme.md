
## Build a Node.js App With Sequelize

#### Installation:

- Clone the project
 ```
 git clone https://github.com/boolfalse/node-codegig.git
cd node-codegig
 ```

- Copy .env.example as .env
- Write appropriate credentials

 ```
npm i
npm start
 ```
